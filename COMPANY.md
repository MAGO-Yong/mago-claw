## 沟通架构

```
老板（当前会话）
  │
  ├─ 牛马（子会话1 / 本会话消息）
  │   └─ 工作记忆 & 工作成果
  │
  ├─ MAGO（子会话2 / 远程会话）
  │   └─ 生活记忆 & 生活成果
  │
  └─ 协调中心
      ├─ 任务队列（TASKS.md）
      ├─ 成果库（RESULTS.md）
      └─ 跨员工通信（通过会话转发）
```

## 任务派发流程

1. 老板在 `TASKS.md` 中新增任务
2. 任务包含：标题、描述、指派对象、deadline、优先级
3. 指派对象（牛马或MAGO）看到任务后执行
4. 完成后在 `RESULTS.md` 中汇报成果
5. 老板验收后标记为完成
