## 🎓 COZE文档目录的设计哲学

### 1. **从定义开始**
- "什么是评测" 是第一个页面
- 不是"快速开始"，而是"概念"

### 2. **分层递进**
```
定义 → 入门教程 → 三个核心对象 → 管理操作 → 高级功能
```

### 3. **三个核心对象**
- 评测集（数据）
- 评估器（规则）
- 实验（结果）

### 4. **两个特殊模块**
- 人工标注（质量保证）
- 轨迹评测（Agent专有）

### 5. **完整的左导航**
- 每个模块都有独立的树形菜单项
- 便于用户快速找到相关文档
