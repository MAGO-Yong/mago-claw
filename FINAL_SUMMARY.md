**两种评估器类型**
- LLM评估器：Prompt驱动，快速初筛（秒级）

### LLM配置参数
```
配置名称、模型提供商、模型名称、API Key（必填）
+
API Endpoint、Temperature、Max Token、Top P、频率限制（可选）
```

### 评估器编辑表单

**HTTP代码评估器**
```
名称 + 评估方式（单选）
+
服务名（必填）、请求路径（必填）
+
Authorization（可选）、每分钟最大请求数
+
输出字段（动态表）
```

**LLM评估器**
```
名称 + 评估方式（单选）
+
LLM选择（必填）、Prompt（必填，支持Markdown）
+
输出字段（动态表）
```

### 自动评估4步向导
```
Step 1: 基本配置（任务名、执行周期）
├─ Step 2: 评估数据（3种方式、数据源、采样率）
├─ Step 3: 评估器（关联评估器、字段映射）
└─ Step 4: 数据回流（目标数据集、回流条件、JsonPath映射）
```

### 数据回流条件
```
支持复杂条件表达式
- 单条件：score >= 2
- 多条件AND：score >= 2 && relevance == 'high'
- 多条件OR：score >= 3 || is_important == true
```
