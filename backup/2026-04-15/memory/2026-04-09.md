### 重要技术洞察（来自官方文档调研）

**安全护栏（LangChain + NeMo + Datadog）**
- 间接 Prompt 注入是最高危攻击面（RAG 召回内容 / Tool 返回值里埋指令）
- NeMo：并行 Rail 编排，5 个护栏并行，延迟仅 +0.5s，检测率 +1.4x
- Datadog：Monitor-Only → Selective Block → Full Block 三阶段上线
- 需建立 Agent Tool 调用行为基线，偏离才能识别多步攻击

**AI 可观测（OTel GenAI SemConv）**
- 应对齐 OpenTelemetry GenAI 语义规范（非自造体系）
- 核心指标：`gen_ai.client.token.usage`（Histogram）/ `gen_ai.client.operation.duration`
- 对齐 OTel 后 AI Span 和 HTTP Span 能在同一 Trace 树共存，真正实现 AI × 基础设施联合分析

**AI 评估（LangSmith）**
- 四类模式：Benchmarking / Unit Tests / Regression Tests / Backtesting
- 关键：数据集需支持"生产 Trace 一键加入数据集"，否则在线↔离线评估闭环打不通
