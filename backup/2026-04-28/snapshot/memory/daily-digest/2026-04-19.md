### 三条演进主线

1. **从"能做"到"可控"**：AutoGPT 能自主但不可控 → LangGraph 人在回路 → 声明式安全策略
2. **从"单干"到"协作"**：单 Agent → CrewAI 多角色 → OpenClaw 平台级多 Agent → 异步子 Agent
3. **从"堆能力"到"管信息"**：更多工具/记忆/技能 → 渐进式披露，精简上下文
