场景 A · SKILL 绑定配置
告警规则 × 排障 SKILL 关联
A1
业务孵化排障 SKILL
业务侧针对高频告警场景，在技能广场创建排障 SKILL（如「新笔记下跌排障」「GC 异常排障」）
🏪 技能广场 xpilot/skill
A2
绑定告警规则
两种方式完成绑定：
① 技能广场页面：SKILL 卡片 → 绑定告警规则 → 搜索/筛选/批量选择
② 对话配置：「帮我查推荐相关告警规则，把 1、3、5 绑定到新笔记下跌排障 SKILL」
页面配置 对话式配置
A3
配置生效 & 验证
绑定完成后，对应告警触发时自动调用关联 SKILL 执行诊断。支持查看绑定关系、测试运行、版本管理
✓ 绑定关系管理 测试运行

场景 B · 实时告警诊断
告警触发 → AI 自动诊断 → Hi IM 推送
B1
告警规则触发
Xray 告警规则命中，生成告警事件，匹配已绑定的排障 SKILL
🔔 告警触发 Xray Rules Engine
B2
Hi IM 推送告警卡片
XRay 助手推送告警事件卡片到值班群，展示告警基础信息，底部显示「AI 诊断中…」状态
📨 Hi IM 卡片 值班群
B3
SKILL 自动执行诊断
后台调用绑定的排障 SKILL，拉取指标/日志/Logview/变更数据，完成根因分析。卡片底部更新为「诊断完成 · 根因 + 置信度」
⚡ SKILL 执行 根因 + 置信度
B4
用户深度排障
点击「接着聊」→ 右侧 AI 对话侧边栏展开，诊断信息作为上下文，用户可继续追问、生成复盘、触发处置动作
💬 Hi IM 侧边栏 处置 & 复盘
📊
场景 C · 每日告警日报
每天 12:00 推送业务线告警分析
C1
定时聚合昨日数据
每天 12:00，按业务线/产品线汇总昨日所有告警事件：总量、级别分布、时段热图、服务分布
⏰ 定时任务 按业务线聚合
C2
AI 根因聚类分析
对所有诊断结果做聚类：TOP 根因分布、重复告警识别、未诊断告警标注、告警风暴检测
🧠 AI 聚类 根因 TOP N
C3
规则优化建议
基于告警数据给出规则优化建议：误报率高的规则（可调阈值）、重复根因（可合并规则）、未覆盖异常（建议新增规则）
📋 优化建议 误报 / 漏报
C4
日报推送到业务群
以结构化卡片形式推送到各业务群，支持点击下钻查看详情，或在 XPILOT 继续追问分析
📨 Hi IM 推送 可交互卡片

场景关系 & 数据流向
A → B
配置完成是实时诊断的前提。SKILL 未绑定的告警规则，触发后无法自动诊断，仅推送基础告警卡片。
B → C
实时诊断产生的根因数据，是日报聚类分析的数据源。诊断覆盖率越高，日报分析越准确。
C → A
日报的规则优化建议，反哺 SKILL 绑定配置的调整（如为高频误报规则解绑 SKILL，或为漏配规则补充绑定）。
